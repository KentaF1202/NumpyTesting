# Numpy Testing

This project comes preloaded with various questions based on numpy's basic functions.
There are various CLI options to customize the questioning.

```bash
-q #For specific question number (mutually exclusive with -a and -r)
-r #For random questions (mutually exclusive with -q)
-a #For all questions (mutually exclusive with -q)
-d #For debug mode
-h #To see all CLI parameters
```
